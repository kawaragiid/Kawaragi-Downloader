# Kawaragi Downloader

Aplikasi downloader sederhana menggunakan yt-dlp dan ffmpeg.

## Fitur
- Download video dari YouTube
- Konversi video ke format MP3

## Instalasi
1. Clone repository ini.
2. Instal dependensi: `pip install -r requirements.txt`.
3. Jalankan aplikasi: `python app/main.py`.

## Persyaratan
- Python 3.8+
- ffmpeg (tersedia di folder `ffmpeg/`)
